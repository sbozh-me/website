# 1.3.14 - Blog Post Grid Layout

## Goal

Add blog post grid below persona card showing latest posts filtered by the selected persona's authors. Simple centered grid with 1-3 cards of equal size (layouts only, no data fetching).

## Deliverables

- [ ] `BlogPostCard` compact card component
- [ ] `BlogPostGrid` centered grid container
- [ ] Section title "Latest Posts"
- [ ] Responsive layout (1/2/3 cards, centered)
- [ ] Placeholder cards for testing
- [ ] Props for author filtering (used in 1.3.16)

### Responsive Behavior

- Mobile: 1 card, centered
- Tablet (md): 2 cards, centered
- Desktop (lg): 3 cards, centered
- All cards same width regardless of count
