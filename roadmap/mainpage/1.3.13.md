# 1.3.13 - Author Carousel Layout

## Goal

Create the author profile card layout with image and carousel foundation for 3 entities with left/right arrow navigation.

## Deliverables

- [x] `AuthorCard` component with avatar, name, title, description, CTAs
- [x] `AuthorCarousel` component with arrow navigation
- [x] Slide transition animation (Framer Motion)
- [x] Active dot indicators (3 dots)
- [x] Keyboard navigation (arrow keys)
- [x] Touch swipe support for mobile
- [x] Author type definition

  name: string
  title: string
  description: string
  avatar: string
