# 1.3.13 - Persona Carousel Layout

## Goal

Create the persona profile card layout with image and carousel foundation for 3 entities with left/right arrow navigation.

## Deliverables

- [ ] `PersonaCard` component with avatar, name, title, description, CTAs
- [ ] `PersonaCarousel` component with arrow navigation
- [ ] Slide transition animation (Framer Motion)
- [ ] Active dot indicators (3 dots)
- [ ] Keyboard navigation (arrow keys)
- [ ] Touch swipe support for mobile
- [ ] Persona entity type definition

## Implementation

### Data Structure

```typescript
// apps/web/types/persona-entity.ts
export interface CTAButton {
  label: string
  href: string
  variant: 'primary' | 'secondary' | 'outline'
  icon?: string
}

export interface PersonaEntity {
  id: string
  name: string
  title: string
  description: string
  avatar: string
  ctaButtons: CTAButton[]
  blogPersonaSlugs: string[] // personas to filter blog posts
}
```
