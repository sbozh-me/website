# 0.8.0 - Projects List Page

## Goal

Create the projects listing page with large image cards for sbozh.me and Discord community projects.

## Deliverables

- [ ] Badge component added to @sbozh/react-ui
- [ ] Project types defined (ProjectStatus, ProjectMeta, ProjectTab, Project)
- [ ] Project data structure with hardcoded projects
- [ ] StatusBadge component (active/beta/coming-soon/archived)
- [ ] ProjectCard component with hero image, status, and meta info
- [ ] ProjectGrid component with responsive layout
- [ ] Projects page updated to use ProjectGrid

## Implementation

### Data Structure

```typescript
// apps/web/lib/projects/types.ts
export type ProjectStatus = 'active' | 'beta' | 'coming-soon' | 'archived';

export interface ProjectMeta {
  label: string;
  value: string;
}

export interface ProjectTab {
  id: string;
  label: string;
  enabled: boolean;
}

export interface Project {
  slug: string;
  title: string;
  tagline: string;
  status: ProjectStatus;
  heroImage: { src: string; alt: string };
  meta: ProjectMeta[];
  tabs: ProjectTab[];
}
```

### Card Design Pattern

Following PostCard styling from blog package:

```tsx
<Link href={`/projects/${slug}`} className="block bg-muted border border-border rounded-lg overflow-hidden transition-colors hover:border-primary">
  <div className="aspect-video relative">
    <Image ... />
    <StatusBadge className="absolute top-4 right-4" />
  </div>
  <div className="p-6">
    <h3>{title}</h3>
    <p className="text-muted-foreground">{tagline}</p>
    <div className="flex gap-4 text-sm text-muted-foreground">
      {meta.map(...)}
    </div>
  </div>
</Link>
```

### Responsive Layout

- Mobile: single column, full-width cards
- Desktop (md+): 2-column grid
