# 0.8.4 - Discord Community Project

## Goal

Populate the Discord community project with full content and demonstrate dynamic tab configuration.

## Deliverables

- [ ] About tab content (community overview, purpose)
- [ ] Motivation tab content (why start a community)
- [ ] Documentation tab content (how to join, rules, channels)
- [ ] Roadmap tab content (community growth plans)
- [ ] Hero image for Discord community project
- [ ] SEO metadata for project detail pages

## Implementation

### Different Tab Configuration

This project demonstrates that each project can have different tabs enabled:

```typescript
// apps/web/lib/projects/data.ts
{
  slug: 'discord-community',
  title: 'Discord Community',
  tagline: 'A place for builders and creators',
  status: 'beta',
  heroImage: {
    src: '/images/projects/discord-community/hero.jpg',
    alt: 'Discord community screenshot',
  },
  meta: [
    { label: 'Members', value: '100+' },
    { label: 'Status', value: 'Growing' },
  ],
  tabs: [
    { id: 'about', label: 'About', enabled: true },
    { id: 'motivation', label: 'Motivation', enabled: true },
    { id: 'documentation', label: 'Documentation', enabled: true },  // Different from sbozh.me
    { id: 'roadmap', label: 'Roadmap', enabled: true },
  ],
}
```

Note: This project uses "Documentation" instead of "Changelog", showing the flexibility of the tab system.
