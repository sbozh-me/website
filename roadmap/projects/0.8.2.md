# 0.8.2 - Project Detail Layout

## Goal

Create the project detail page structure with sidebar navigation (desktop) and horizontal scroll tabs (mobile).

## Deliverables

- [ ] Project layout with sidebar and content area
- [ ] Desktop sidebar with vertical tab navigation
- [ ] Mobile horizontal scroll tabs
- [ ] Nested routing: `/projects/[slug]/[tab]`
- [ ] Default route redirect to first enabled tab

## Implementation

### Routing Strategy

Nested routes for SEO granularity:

```
/projects/sbozh-me          → redirects to /projects/sbozh-me/about
/projects/sbozh-me/about    → About tab content
/projects/sbozh-me/motivation → Motivation tab content
```

### Layout Structure

```
Desktop (lg+):
┌─────────────────────────────────────┐
│ ProjectHeader (hero + title)        │
├──────────┬──────────────────────────┤
│ Sidebar  │ Tab Content              │
│ - About  │                          │
│ - Motiv. │                          │
│ - Change │                          │
│ - Road.  │                          │
└──────────┴──────────────────────────┘

Mobile:
┌─────────────────────────────────────┐
│ ProjectHeader                       │
├─────────────────────────────────────┤
│ [About] [Motivation] [Changelog] →  │
├─────────────────────────────────────┤
│ Tab Content                         │
└─────────────────────────────────────┘
```

### Mobile Tab Scroll

```tsx
<nav className="lg:hidden overflow-x-auto scrollbar-hide">
  <div className="flex gap-2 px-4 py-3">
    {tabs.filter(t => t.enabled).map(tab => (
      <Link
        href={`/projects/${slug}/${tab.id}`}
        className={cn(
          "px-4 py-2 rounded-full text-sm whitespace-nowrap transition-colors",
          isActive ? "bg-primary text-primary-foreground" : "bg-muted text-muted-foreground hover:bg-muted/80"
        )}
      >
        {tab.label}
      </Link>
    ))}
  </div>
</nav>
```

### Desktop Sidebar

```tsx
<aside className="hidden lg:block w-[250px] sticky top-24">
  <div className="space-y-2">
    {tabs.filter(t => t.enabled).map(tab => (
      <Link
        href={`/projects/${slug}/${tab.id}`}
        className={cn(
          "block px-4 py-2 rounded-md transition-colors",
          isActive ? "bg-primary/10 text-primary" : "text-muted-foreground hover:text-foreground"
        )}
      >
        {tab.label}
      </Link>
    ))}
  </div>
</aside>
```
