## Goal

Add JSON-LD structured data for better search engine understanding and rich snippets.

## Deliverables

- [ ] Article schema for blog posts

- [ ] BreadcrumbList for navigation

## Implementation

### Blog Post - Article Schema

```typescript
// apps/web/app/(main)/blog/[slug]/page.tsx

// Add to the page component
const jsonLd = {
  '@context': 'https://schema.org',
  '@type': 'Article',
  headline: post.title,
  description: post.excerpt,
  image: post.image?.src,
  datePublished: post.date,
  author: {
    '@type': 'Person',
    name: post.persona.name,
  },
  publisher: {
    '@type': 'Organization',
    name: 'sbozh.me',
    logo: {
      '@type': 'ImageObject',
      url: 'https://sbozh.me/android-chrome-512x512.png',
    },
  },
};

// In JSX:
<script
  type="application/ld+json"
  dangerouslySetInnerHTML={{ __html: JSON.stringify(jsonLd) }}
/>
```

### Breadcrumb Schema (Optional)

```typescript
// Helper function for breadcrumbs
function generateBreadcrumbSchema(items: { name: string; url: string }[]) {
  return {
    '@context': 'https://schema.org',
    '@type': 'BreadcrumbList',
    itemListElement: items.map((item, index) => ({
      '@type': 'ListItem',
      position: index + 1,
      name: item.name,
      item: item.url,
    })),
  };
}

// Usage in blog post:
const breadcrumbJsonLd = generateBreadcrumbSchema([
  { name: 'Home', url: 'https://sbozh.me' },
  { name: 'Blog', url: 'https://sbozh.me/blog' },
  { name: post.title, url: `https://sbozh.me/blog/${post.slug}` },
]);
```

## Helper Component

Create a reusable component for JSON-LD:

```typescript
// apps/web/components/JsonLd.tsx
export function JsonLd({ data }: { data: Record<string, unknown> }) {
  return (
    <script
      type="application/ld+json"
      dangerouslySetInnerHTML={{ __html: JSON.stringify(data) }}
    />
  );
}
```
