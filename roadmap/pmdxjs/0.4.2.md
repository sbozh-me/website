## Component Specs

### Header

```typescript
interface HeaderProps {
  name: string;
  subtitle?: string;
  contact?: string[]; // ["email", "phone", "location"]
}
```

### Entry

```typescript
interface EntryProps {
  company: string;
  role: string;
  dates: string;
  location?: string;
  children: React.ReactNode; // Bullet points
}
```

### Tags

```typescript
interface TagsProps {
  items: string[];
}
```

### Section

```typescript
interface SectionProps {
  title: string;
  children: React.ReactNode;
}
```

## Styling

All components use Tailwind classes with Obsidian Forge color tokens:
- Primary: `#8b5cf6` (purple)
- Secondary: `#f59e0b` (amber)
- Accent: `#22c55e` (green)

## Demo

```typescript
const cvSource = `
:::page
# John Doe
subtitle: Senior Engineer
contact: john@example.com | +1-555-0100

## Experience

:::entry ACME Corp | Lead Developer | 2020-Present | SF
- Led team of 8 engineers
- Increased performance by 40%
:::

## Skills
#tag TypeScript
#tag React
:::page-end
`;

const element = render(cvSource);
// Full CV with styled header, entries, and tags
```
